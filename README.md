# Hack4Freedom Lagos 2026

📍 Lagos, Nigeria  
🗓 April/May 2026
👩🏽‍💻 Women-focused Bitcoin & freedom tech hackathon  

Hack4Freedom Lagos 2026 is part of the Hack4Freedom hackathon series, a women-focused program building open-source freedom technology across the Global South.

This repository serves as the **central source of truth** for the Lagos 2026 cohort — including program information, resources, speakers, mentors, and projects built during the hackathon.

---

## About Hack4Freedom Lagos

Hack4Freedom Lagos is a **two-week hybrid hackathon** focused on building open-source tools using **Bitcoin, Lightning, Nostr, and ecash**.

The program combines:
- Education and technical workshops  
- Hands-on mentorship  
- Public, open-source development  

Participants will work in teams to build solutions addressing real local challenges, with an emphasis on learning, collaboration, and long-term impact.

---

## Program Structure

### Week 1 — Education & Ideation
- Bitcoin fundamentals and freedom tech context  
- Lightning, Nostr, and ecash workshops  
- Open-source workflows and GitHub onboarding  
- Team formation and problem discovery  
- Mentor-led technical sessions  

### Week 2 — Building & Mentorship
- Daily stand-ups and progress check-ins  
- Mentor office hours  
- Focused development sessions  
- Project iteration and testing  
- Demo Day preparation
